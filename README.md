# Line_Circle_Generation_Algorithm  
The project is a program for generating a line or circle based on specific algorithms listed below:
## Line Generation Algorithm:  
* DDA  
* Bressenham  
## Circle Generation Algorithm:  
* Modular  
* Midpoint
